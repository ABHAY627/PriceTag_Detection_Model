# Retail Price Tag Detection & OCR

A full-stack retail operations tool that detects price tags on shelf images
and extracts numeric prices using computer vision and OCR.

---

## Project Structure

```
MUSE_ASSIGNMENT/
├── data/
│   ├── raw/              ← Drop your images + YOLO .txt labels here
│   ├── annotated/        ← (optional) post-annotation exports
│   ├── augmented/        ← auto-generated by augment_dataset.py
│   ├── splits/           ← auto-generated train/val/test split
│   └── dataset.yaml      ← YOLO dataset config
├── models/
│   ├── checkpoints/      ← saved .pt weight files
│   └── exports/          ← ONNX / TorchScript exports
├── backend/              ← FastAPI service
├── frontend/             ← Next.js + TypeScript + Tailwind
├── scripts/
│   ├── prepare_dataset.py
│   └── augment_dataset.py
├── notebooks/            ← exploratory notebooks
├── requirements.txt
├── DATA_REPORT.md        ← auto-generated after prepare_dataset.py
├── DETECTION_REPORT.md   ← auto-generated after training
└── PROGRESS.md           ← checkpoint tracker
```

---

## Quickstart

### 1. Install Python dependencies
```bash
pip install -r requirements.txt
```

### 2. Add your data
Drop shelf images and YOLO-format `.txt` label files into `data/raw/`.

### 3. Prepare & augment dataset
```bash
python scripts/prepare_dataset.py
python scripts/augment_dataset.py
```

### 4. Train (Checkpoint 2 — coming after data review)
```bash
python scripts/train_detector.py
```

### 5. Run backend (Checkpoint 4)
```bash
uvicorn backend.app.main:app --reload
```

### 6. Run frontend (Checkpoint 5)
```bash
cd frontend && npm run dev
```
